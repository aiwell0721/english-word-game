# 小学生英语单词学习游戏

一个专为小学生设计的互动式英语单词学习游戏，通过游戏化机制让学习变得有趣高效。

## 🎯 项目简介

本项目通过游戏化的方式帮助小学生学习英语单词，包含丰富的互动功能、成就系统和社交元素。

## ✨ 核心功能

### 基础学习功能
- **单词学习**：426个精选词汇，覆盖6个等级
- **多种学习模式**：拼写练习、选择题、听音辨词、图片匹配
- **智能复习**：基于艾宾浩斯遗忘曲线的复习提醒系统

### 游戏化系统
- **积分系统**：完成学习任务获得积分
- **成就系统**：解锁各种学习成就徽章
- **排行榜**：实时排行榜，与好友比拼学习进度
- **每日任务**：每日学习任务，养成学习习惯

### 社交功能
- **好友系统**：添加好友，互相监督学习
- **分享功能**：分享学习成果到微信、QQ、微博
- **学习对比**：查看与好友的学习进度对比

### 家长管理
- **家长认证**：家长身份验证系统
- **学习监控**：实时监控孩子的学习情况
- **学习报告**：生成详细的学习进度报告
- **进度对比**：对比不同时间段的学习效果

## 🏗️ 技术架构

### 前端技术栈
- **框架**：React 18 + TypeScript
- **路由**：React Router v6
- **状态管理**：Context API + Hooks
- **UI组件库**：Tailwind CSS
- **测试**：Vitest + React Testing Library

### 后端技术栈
- **框架**：Express.js + TypeScript
- **数据库**：SQLite（开发） / PostgreSQL（生产）
- **认证**：JWT
- **API文档**：Swagger/OpenAPI

### 开发工具
- **包管理**：pnpm
- **代码规范**：ESLint + Prettier
- **版本控制**：Git

## 🚀 快速开始

### 环境要求
- Node.js >= 18.x
- pnpm >= 8.x
- PostgreSQL >= 14.x（生产环境）

### 安装步骤

1. **克隆仓库**
```bash
git clone https://github.com/你的用户名/english-word-game.git
cd english-word-game
```

2. **安装依赖**
```bash
# 前端
cd frontend
pnpm install

# 后端
cd ../backend
pnpm install
```

3. **配置环境变量**
```bash
# 复制环境变量模板
cp .env.example .env

# 编辑.env文件，配置数据库等
```

4. **初始化数据库**
```bash
psql -U postgres -f database/schema.sql
psql -U postgres -f database/seed.sql
```

5. **启动开发服务器**
```bash
# 前端（端口3000）
cd frontend
pnpm dev

# 后端（端口5000）
cd ../backend
pnpm dev
```
