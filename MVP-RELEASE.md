## 📱 安装部署

### 开发环境

```bash
# 克隆项目
git clone [repository-url]
cd english-word-game

# 安装前端依赖
cd frontend
npm install

# 启动前端开发服务器
npm run dev

# 安装后端依赖
cd ../backend
pip install -r requirements.txt

# 启动后端服务器
python app.py
```

### 生产环境

```bash
# 前端构建
cd frontend
npm run build

# 部署构建产物
# dist/ 目录可以部署到任何静态文件服务器
```

---

## 🔧 配置说明

### 环境变量

创建 `.env` 文件：

```env
# 后端配置
FLASK_APP=app.py
FLASK_ENV=production
SECRET_KEY=your-secret-key
DATABASE_URL=sqlite:///word_game.db

# 前端配置
VITE_API_URL=http://localhost:5001
```

### 数据库初始化

```bash
# 初始化数据库
cd backend
python app.py

# 导入词汇数据
python database/import_words.py
```

## 📖 使用指南

### 学生端

1. **注册/登录**
   - 首次使用需要注册账号
   - 输入用户名和密码
   - 选择年级（1-6年级）

2. **开始学习**
   - 选择学习模式（拼写/选择题/听音/图片匹配）
   - 选择学习等级
   - 开始答题

3. **查看进度**
   - 查看学习统计
   - 查看错题本
   - 查看学习周报

4. **游戏化**
   - 查看积分和等级
   - 解锁成就
   - 完成每日任务
   - 查看排行榜

### 家长端

1. **家长注册**
   - 使用家长专用账号注册
   - 关联子学生账号

2. **监控学习**
   - 查看子学生的学习进度
   - 查看学习报告
   - 对比学习数据

3. **设置限制**
   - 设置每日学习时长
   - 设置每周单词目标
   - 开启/关闭通知
